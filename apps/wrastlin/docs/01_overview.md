# Wrestling Legacy Game — Overview

## Concept
This project is an asynchronous legacy-style social wrestling management game. Each player acts as the **manager of an AI-controlled wrestler** rather than controlling a character directly.

The game runs on a **weekly cycle**:

1. Managers interact with their wrestler during the week.
2. Managers submit requests and influence the show storyteller.
3. The system generates a weekly wrestling show.
4. Results update wrestler state and the persistent world.
5. The next week begins.

## Core Fantasy
Players are managing volatile AI personalities. Wrestlers may follow, distort, or ignore manager advice depending on trust, personality, and emotions.

## Core Pillars

### Persistent Characters
Each wrestler has:
- personality
- emotional state
- relationships
- memories

### Emergent Storylines
Rivalries evolve organically based on match results, promos, betrayals, and humiliations.

### Weekly Event Cycle
Players participate asynchronously during the week, then gather to watch the generated show.

### AI as Presentation Layer
AI generates dialogue, promos, commentary, and narration. Structured game logic determines outcomes.
