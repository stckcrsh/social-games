# Weekly Game Loop

## Phase 1: Week Start
The system opens a new week and resets submission windows.

Managers can:
- talk to their wrestler
- submit guidance
- submit storyteller requests
- place bets

## Phase 2: Manager Interaction
Managers interact with their wrestler through a chat-like interface.

Advice can influence:
- emotional state
- trust
- match style
- rivalry escalation

## Phase 3: Weekly Submission
Managers submit structured requests to the storyteller.

Examples:
- request a promo
- challenge a rival
- request a title match
- escalate a feud

Managers can attach money to increase influence.

## Phase 4: Show Generation
At the scheduled show time:

1. The storyteller loads world state.
2. Manager requests are evaluated.
3. A show card is assembled.
4. Segments and matches are resolved.
5. The show script is generated.

## Phase 5: Post Show Updates
The system updates:

- wrestler emotions
- wrestler memories
- rivalries
- titles
- betting payouts

The next week then begins.
