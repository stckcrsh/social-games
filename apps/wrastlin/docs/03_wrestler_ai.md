# Wrestler AI System

Each wrestler is represented by an AI-driven character agent.

## Personality Traits
Example traits:

- ego
- anger
- honor
- loyalty
- ambition
- discipline
- showmanship

These traits change slowly or remain static.

## Emotional State
Emotions change week to week.

Example emotions:

- confidence
- frustration
- fatigue

Emotions influence decision-making and match behavior.

## Relationships
Wrestlers track relationships with others.

Relationship values represent:

- hatred
- respect
- trust

These influence alliances, betrayals, and match intensity.

## Memory System
Important events are stored as memories.

Example memory types:

- humiliation
- betrayal
- victory
- injury
- promo insult

Each memory stores:
- week
- source
- intensity
- emotional tag

Memories influence future behavior and promos.

## Manager Trust
Each wrestler tracks trust toward their manager.

Trust affects how likely advice is followed.

Ranges:
- low trust → advice ignored
- medium trust → advice partially followed
- high trust → advice often followed
